## 🍯 풀이  
- 순서대로 정렬하여 중앙에 있는 인덱스를 출력한다.  

**내 코드**  
```python
def solution(array):
    array = sorted(array)
    return array[len(array) // 2]
```

**다른 분 코드**  
: 코드를 좀 더 간결하게 작성해야겠다.

```python
def solution(array):
    return sorted(array)[len(array) // 2]
```
