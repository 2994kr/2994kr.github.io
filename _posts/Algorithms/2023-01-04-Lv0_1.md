## 🍯 풀이  
- 분수를 합하기 위해 분모끼리 곱한 값에 맞추어 계산한다.
- 최대공약수를 구하는 math.gcd 함수를 사용한다.

```python
import math

def solution(denum1, num1, denum2, num2):
    num = num1 * num2
    denum = denum1 * num2 + denum2 * num1

    gcd_value = math.gcd(num, denum)

    return [denum / gcd_value, num / gcd_value]
```
